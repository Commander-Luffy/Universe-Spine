The 7 axioms (A0 VOID → A6 OPACITY) + Theorem 1 (BECAUSE) + Theorem 2 (Identity Completeness) live in the DPT physics repo. ipv8 infra references them for protocol design but does not own them.

## Quick Reference

```
A0 VOID        — does something exist?
A1 DISTINCTION — is something different?
A2 ROTATION    — can something turn?
A3 DOUBLING    — can one become two?
A4 ENGINE      — can two combine into something new?
A5 TOTALITY    — does everything add up?
A6 OPACITY     — can you tell from inside?
```

## How ipv8 Uses the Axioms

- A0 → address 0.0.0.0 (pre-existence, unborn)
- A1 → T/F binary, the ! cut → firewall ACCEPT/DENY
- A2 → 6+2 encoding rotation bits, phase carrier
- A3 → τ(N) = 2^N → byte=8, nibble=4, bit=1
- A4 → I/CC/U/SU control loop → 7 security gates
- A5 → convergence → BGP path selection, route stability
- A6 → opacity → no level self-verifies → layer separation
